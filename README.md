🍽️ Food For All – Food Donation Platform

Food For All is a web platform that connects food donors with people in need.
The goal of this project is to reduce food waste and help feed hungry people by allowing individuals, restaurants, and stores to donate surplus food.

🌍 Project Overview

Food For All allows users to:

Donate surplus food

View available food donations

Locate donation points on a map

Help reduce food waste

Support communities in need

This platform is designed to be simple, fast, and accessible for donors and volunteers.

🚀 Features
🧾 Food Donation Form

Users can submit food donations with details like:

Donor name

Food type

Quantity

Description

Location

Payment (optional support)

📍 Interactive Map

Displays donation locations

Built using Leaflet.js

Helps volunteers locate food easily

📦 Donation Listing

Shows available food donations with:

Food type

Quantity

Donor name

Location

Time posted

✨ Animated UI

Smooth scroll animations

Floating food particles

Modern UI design

📊 Live Counters

Animated statistics showing platform activity.
